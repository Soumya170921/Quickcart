# QuickCart 🛒✨

> A lit, open-source Next.js + Tailwind e-commerce front-end. Scroll, click, cart — all the vibes. Contributors welcome!

---

##  What’s the Tea?

- Built with **Next.js** + **Tailwind CSS** — modular AF, retina-ready, and lightning-fast.
- Fully **responsive** for every screen size — phone, tablet, laptop, you name it.
- **Reusable UI components** for days — sliders, cards, navbars… you do you.
- Flexible layouts & colors — make it your own.
- Open for contributions: UI love, animations, new pages, themes… come collab!
